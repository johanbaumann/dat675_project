To run Random Forest models on a dataset, run

> python src/RF_model/RF_fixed_predictor.py

from the project root.

RF hyper parameters can be changed in the RF_fixed_predictor.py file, aswell as which dataset it will train and validate on. To change the dataset, change  GROUP_FOLDER to the folder of the desired dataset in the data directory.
